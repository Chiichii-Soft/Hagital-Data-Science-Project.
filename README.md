# Data Science Project | Hagital Consulting

## Project Overview
An advanced data science and exploratory data analysis (EDA) project executed at Hagital Consulting to transform raw data into actionable business insights. 

## Core Responsibilities & Execution
* **Data Processing:** Cleaned, structured, and prepared complex datasets for optimal analysis.
* **AI-Assisted Operations:** Leveraged advanced composition and data operations to streamline code documentation and data flows.
* **Analytical Insights:** Engineered data visualizations and statistical summaries to uncover underlying trends.

## Technologies Used
* **Jupyter Notebook** (Python)
* **Pandas / NumPy** (Data Manipulation)
* **Matplotlib / Seaborn** (Data Visualization)

## Project Deliverables
* `Hagital-Data-Science-Project`: The complete, public repository housing the full case study.
* `Lagos_Rides.ipynb`: The primary, fully documented Jupyter Notebook file analyzing the ride dataset.
